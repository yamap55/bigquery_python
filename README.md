# python_repository_simple

本リポジトリは BigQuery を Python で触るリポジトリです

## 環境詳細

- Python : 3.9

### 開発手順

1. VS Code 起動
2. 左下の緑色のアイコンクリック
3. 「Remote-Containersa: Reopen in Container」クリック
4. しばらく待つ
   - 初回の場合コンテナー image の取得や作成が行われる
5. 起動したら開発可能
